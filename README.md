# Rasti Lari marketing site (Next.js + TypeScript + Tailwind + shadcn-style UI + Framer Motion)

This is a starter project for the Rasti Lari Group website built with:

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Simple shadcn-style UI components (Button, Card, Badge)
- Framer Motion animations

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Open `http://localhost:3000` in your browser.

### Main pages

- `/` – Home
- `/about` – About us & history
- `/products` – Product info listing with categories and search
- `/products/[slug]` – Individual product detail with additional information, customer testimonials and related products
- `/gallery` – Image gallery
- `/food` – Food & warehouse information with a "View more" button linking to food products
- `/contact` – Contact page with map `<iframe>`
- `/blog` – Simple static blog layout

The **E‑Shop** item in the main navigation links externally to `https://shop.rastilari.com/`.
